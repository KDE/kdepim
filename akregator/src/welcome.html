Welcome to aKregator

Use the tree on the left to manage your RSS feeds.

  • Right-click a feed group (e.g. All Feeds) and choose:
    • Add to add a new feed via the Add Feed dialog
    • Add Folder to add a folder, that helps you organize your feeds
    • Delete to remove existing feed or folder
    • Modify to edit existing feed or folder
    • Fetch to fetch this feed. When clicked on a feed group will fetch from all feeds in this folder and subfolders.

Click "Fetch All" to refresh all added feeds
