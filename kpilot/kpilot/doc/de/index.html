About KPilot v1.1b

KPilot is software for syncing the 3Com Palm Pilot and IBM Workpad (UNTESTED!) with a machine running some flavor of unix. The actual PC->PP communcations was originally based on pilot-link.0.8.7. This may or may not change as KPilot develops.

This is the second release of KPilot, The have been a few bug fixes (most notably the compile problem a lot of people were experiencing), some cosmetic changes, and signifcant rewrite and expanding of the syncing capabilities.

Changes

Version 1.1b

  • SmartSync backing up of modified data even if an unrecognized database type.
  • Import/Export to/from virtually any text format for Addresses.
  • Import/Export to/from text files for memos.
  • Rewrite and expansion of danduit (KPilot's answer to conduits) interface to make it easier for others to add other danduits.
  • Docs!

Version 1.0b

  • Full Backup/Restore capabilities of Palm Pilot
  • File (prc,pdb) installation with drag and drop support
  • Email Synchronization (via POP3 and sendmail)
