The View Menu

View->Select View

Lists all available views.

View->Add View...

Opens a dialog for creating a new view.

View->Modify View...

Opens a dialog where you can modify the settings of the current view.

View->Delete View

Deletes the current view.

View->Contact Editor

Display or hide the contact editor window.

View->Distribution List Editor

Display or hide the distribution list editor window.

View->Address Books

Display or hide the adressbook window.

View->Refresh View

Refreshes the current view.
