4. Menus and Keyboard Shortcuts

Each menu item is discussed below. When there is a keyboard shorcut that performs a menu item function, it is listed after the menu item.

4.1 The Main Mail Reader window

File menu

New Composer (Ctrl+N)

Creates a new Composer window.

New Mail Reader

Creates a new Mail Reader window.

Save As... (Ctrl+S)

Saves the currently displayed message.

Print... (Ctrl+P)

Prints the currently displayed message.

Check Mail (Ctrl+L)

Checks mail in the active account.

Check Mail in... submenu

Submenu that lets you check mail from a particular account.

Settings...

Opens the Settings window. You can configure the settings for your identity, accounts, network, PGP, the appearance of the composer and mail reader windows, and various other settings here.

Addressbook...

Opens up the Addressbook.

Filters...

Opens the Filters window.

Close (Ctrl+W)

Closes current Mail Reader window.

Quit (Ctrl+Q)

Exits KMail.

Edit menu

Copy (Ctrl+C)

Copies selected text to the clipboard.

Folder menu

Create...

Opens the New Folder dialog which lets you create new folders

Modify...

Opens up the Modify Folder dialog which lets you change the settings for the selected folder.

Compact

Compacts the folder file to reduce its disk space usage. Use this command if your mailboxes seem to be much larger than they ought to be.

Empty

Moves all of the messages in the selected folder into the trash folder. If the selected folder is the trash folder, the messages are permanently deleted.

Remove

Removes the selected folder.

Message menu

Next (N)

Selects the next message in the Headers pane.

Next Unread (+)

Selects the next unread message in the Headers pane.

Previous (P)

Selects the previous message in the Headers pane.

Previous Unread (-)

Selects the previous unread message in the Headers pane.

Reply... (R)

Opens up the Composer window with the contents of the selected message's Reply-To field in the To: field.

Reply All... (A)

Opens up the Composer window and copies the contents of the selected message's Reply-To and To fields into the To: field and the cc: and bcc: fields into the cc: and bcc: fields of the Composer window.

Forward (F)

Forwards the message to a new recipient.

Edit (T)

Edits the selected message if it is editable.

Set Status submenu

Allows you to change the status of the selected message to one of the below:

  • New
  • Unread
  • Read
  • Replied
  • Queued
  • Sent

Mark All (K)

Selects all of the messages in the current folder.

Move (M)

Moves all of the selected messages to a new folder.

Copy (S)

Copies the selected messages into a new folder.

Delete (D)

Moves the selected messages into the trash folder.

Send Again...

Sends the message again

Apply Filters (Ctrl+J)

Applies your filters to the selected messages

View Source...

Views the message and its complete headers in plain text format in a new window.

View menu

Brief, Fancy, Standard, Long, and All Headers

Changes the number of headers that are displayed and the formatting of the headers in the Message pane.

Iconic, Smart, and Inlined Attachments

Changes the way that attachments appear in the Message pane.

Help menu

Contents (F1)

Invokes the KDE help system with the KMail help pages displayed.

About Kmail...

Displays essential information about KMail.

About KDE...

Displays information about KDE.

4.2 The Composer window

File menu

Send (Ctrl+Enter)

Queues the message in the outbox for sending it later.

Send Now

Sends the message immediately.

Insert File...

Inserts the selected text file into the message, starting at the cursor.

Address Book

Opens up the address book.

Print

Prints the message.

New Composer (Ctrl+N)

Opens up a new Composer window.

New Mail Reader

Opens up a new Mail Reader window.

Close (Ctrl+W)

Closes the Composer window.

Edit menu

Cut (Ctrl+X), Copy (Ctrl+C>, Paste (Ctrl+V)

These items are for cutting, copying, and pasting text. They work like in most editors.

Mark all

Selects all of the text in your message.

Find (Ctrl+F)

Finds words, phrases, or expressions in the message.

Replace (Ctrl+R)

Lets you replace words, phrases, or expressions in your message with other words or phrases.

Spellcheck...

Uses KSpell to check the spelling in the body of your message.

Options menu

Urgent

Sets the priority of the message to Urgent.

Confirm Delivery

If you choose this option, you will receive a confirmation email once your message is successfully delivered to its destination mail server.

Confirm Read

If you choose this option, you will receive a confirmation email once your message is downloaded and read by its recipient.

Spellchecker...

Allows you to configure Kspell, KDE's spellchecker.

View menu

This menu lets you toggle the display of the header fields in the composer window.

Help menu

Contents (F1)

Invokes the KDE help system with the KMail help pages displayed.

About Kmail...

Displays essential information about KMail.

About KDE...

Displays information about KDE.
