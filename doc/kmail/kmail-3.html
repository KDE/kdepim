3. Fundamentals

The main screen is devided into 3 parts:

1. The folder widget (upper left widget).

2. The mail header widget (upper right widget).

3. The reader widget (lower widget).

Clicking a folder will open it and display the messages in the header widget. Clicking one of the headers will make the message appear in the reader widget.

3.1 Folders and Accounts

KMail allows you to have multiple folders and accounts. The philosophy is that you have certain accounts connected to certain folders so if you open a folder and try to get new mail it will automatically open the connection to the bouned account. You can easily create, modify & delete folders and accounts via the menubar items.
