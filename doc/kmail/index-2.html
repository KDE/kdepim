2. How to get started...

This will be a very short introduction to kmail and its usage so you can start working with it right away! For more in depth information see the Features section.

2.1 KMail's first invokation....

Invoking kmail for the first time it will create a Directory and some files in your home directory. This is the " /Mail" directory which holds the initial folders (outbox,sent-mail,trash & inbox).

Ok,looks good...what now?

The main screen is devided into 3 parts:

1. The folder widget (upper left widget).

2. The mail header widget (upper right widget).

3. The reader widget (lower widget).

Clicking a folder will open it and display the messages in the header widget. Most likely there won't be any messages visible on kmail's first start. Clicking one of the headers will make the message appear in the reader widget.

Setting up kmail

Please select "Settings" from the "File" - menu. In this Tabdialog you now have to enter some initial information so kmail can work correctly. In addition you can specify some preferences to adjust kmail to your needs.
